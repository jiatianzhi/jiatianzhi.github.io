## 快速开始

- 安装nodejs
- 安装docsify工具包：(sudo) npm i docsify-cli -g
- 初始化目录结构：docsify init ./docs
- 本地预览：docsify serve docs

- index.html是网站入口（相关的配置大多写在这里）
- README.md 会作为主页内容渲染
- .nojekyll
